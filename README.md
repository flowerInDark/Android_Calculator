# Android_Cauculator
# 安卓简易计算器（java）
我的第一个安卓小app
## 功能

原理**中缀转后缀**，利用后缀计算，有检查计算式是否合法功能，

支持加减乘除运算，负数与小数运算，大数和高精度运算.//由于精度问题，部分小数点后数字不正确

支持当"+-"、"--"、"*-"、"/-" 四种情况时，视为对一个负数进行加减乘除，且支持（-n）格式的负数

## UI
Button带有**点击变色**功能，**字体自适应显示框改变大小**功能。
